## opencv 配置

- 项目配置-VC++目录-包含目录

```bash
D:\opencv\opencv\build\include
D:\opencv\opencv\build\include\opencv2
```

- 项目配置-VC++目录-库目录

```bash
D:\opencv\opencv\build\x64\vc15\lib
```

- 项目配置-链接器-输入-附加依赖项

```bash
opencv_world343.lib
opencv_world343d.lib
```
