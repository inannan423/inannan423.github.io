## 安装 Vue CLI

### 控制台安装

```bash
npm install vue-cli -g
```

### 检查版本

```bash
vue -version
```

## 新建项目

```bash
vue init webpack ProjectName
```

**接着按照引导配置完成设置即可**。  

## 运行项目

### 依赖安装

```bash
npm install
```

### 在浏览器运行

```bash
npm start
```

### 打包

```bash
npm run build
```
